Security

Protecting your data and assets

At PMT Markets, security is our top priority. We employ multiple layers of protection to safeguard your information and assets.

Encryption

All data transmitted to and from our platform is encrypted using industry-standard TLS 1.3 encryption.

Secure Storage

User data is stored in secure, access-controlled environments with regular security audits and monitoring.

Multi-Factor Authentication

We support MFA to provide an additional layer of security for user accounts.

Bug Bounty Program

We maintain a bug bounty program to encourage security researchers to report vulnerabilities.

Security Contact

Report security issues to: security@pmtmarkets.com
